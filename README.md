# macropad

A custom 3x3 mechanical macropad based on the Seed Studio XIAO RP2040.

## Features

- 9-key mechanical macro pad
- XIAO RP2040 microcontrollre
- KMK firmware (CircuitPython)
- USB-C connectivity
- Custom PCB designed in KiCad
- Custom 3D-printed enclosure
- MX-compatible switches
